# Kojo Hub | Trench War

A Roblox script made for [Trench War](https://www.roblox.com/games/6654918151/Trench-War).
Join our [Discord](https://discord.gg/wRJ6TXfSJQ) to get the key and ask questions.


For the Targeting tab to work, you must be **Spawned** and have the **Pistol** tool in your inventory. You can have it equipped or unequipped. Both will work.
